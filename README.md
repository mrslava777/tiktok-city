# ⚔️ Boss vs Viewers

Мобильное веб-приложение для **TikTok Live**. На экране — огромный босс с большим запасом HP. Зрители стрима **подарками наносят урон** и соревнуются за титул **MVP**, за **последний удар** и место в **Зале Славы**.

Стек: **Node.js + Express + Socket.IO + TikTok Live Connector**. Без базы данных — всё хранится в памяти.

---

## 🎮 Что есть

- 🐉 **Босс** со случайным именем, видом, фоном и запасом HP (100 000 / 250 000 / 500 000 / 1 000 000).
- 🎁 **Урон от подарков** — настраивается в `config.js` (Rose 10, Heart 25, TikTok 50, средние 100, большие 500, дорогие 2000, супер 5000).
- 💥 **Криты** — случайный x2/x5 с всплывающими цифрами и тряской босса.
- 👑 **MVP** — игрок с наибольшим уроном текущему боссу. Серия MVP растёт, если держит лидерство (x5, x10, x20…). Анимация при смене лидера.
- ⚔️ **Последний удар** — добивший получает титул «Убийца босса», полноэкранная победа, затем 15 сек до нового босса.
- 🏆 **Топ-10** игроков и 📜 **живая лента** событий в реальном времени.
- 🏛️ **Зал Славы** — последние 50 убийц боссов.
- 🎚️ **Уровни** по суммарному урону за эфир: Новичок → Воин → Рыцарь → Герой → Легенда → Бог Битвы.

---

## 🚀 Запуск

```bash
npm install
npm start
```

Открой `http://localhost:3000` (на сервере — публичный адрес хостинга).

## 🔧 Подключение к TikTok-лайву

Два способа:

1. **В интерфейсе** — нажми ⚙️, введи `@username` идущего эфира и «Подключить».
2. **Через конфиг** — в `config.js` задай `TIKTOK_USERNAME`, или переменную окружения:
   ```bash
   TIKTOK_USERNAME=tvoy_nick npm start
   ```
   Тогда приложение подключится к лайву автоматически при старте.

> Подключение работает **только когда аккаунт реально в эфире** (TikTok Live запущен). Иначе появится подсказка «Аккаунт сейчас не в эфире».

Нет эфира под рукой? Жми **🎲 Тест-удар** — приложение сгенерирует случайные подарки, чтобы проверить всю механику.

## ⚙️ Настройка (всё в `config.js`)

- `GIFT_DAMAGE` — урон по подаркам (по имени или по стоимости в алмазах).
- `CRIT` — вероятность крита и доля x5.
- `BOSSES` / `BOSS_HP_POOL` — список боссов и возможные значения HP.
- `RESPAWN_SECONDS` — отсчёт до нового босса (по умолчанию 15).
- `LEVELS` — пороги титулов игроков.
- `MVP_STREAK_INTERVAL_MS` — как часто растёт серия MVP (по умолчанию раз в минуту).

После правок в `config.js` перезапусти сервер.

## 📂 Структура

```
config.js          — все настройки (урон, боссы, уровни, криты, таймеры)
game.js            — игровой движок (босс, игроки, MVP, последний удар, Зал Славы)
tiktok.js          — обёртка над TikTok Live Connector
server.js          — Express + Socket.IO, REST API, трансляция событий
public/index.html  — интерфейс
public/style.css   — мобильный игровой дизайн
public/app.js      — клиент: сокеты и анимации
```
