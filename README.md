# 👑 King of Live

Мобильное веб-приложение для **TikTok Live**. На эфире существует только один **Король**. Любой зритель может захватить трон. Пока игрок — Король, остальные зрители (Охотники) пытаются его свергнуть подарками. Чем дольше Король держит трон — тем больше **награда за его голову**. Кто наносит последний удар — становится новым Королём.

Реальное время через **Socket.IO**. Без базы данных — все данные в памяти.

---

## 🎮 Правила

- **Старт эфира:** первый зритель, отправивший подарок, становится **Королём** (HP 10 000, награда 1000, таймер трона 00:00). Все остальные — **Охотники**.
- **Атака:** охотники бьют Короля подарками. Каждый подарок наносит урон (бывают криты ×2 / ×5).
- **Король пал:** когда HP = 0 — полноэкранная анимация «⚔️ КОРОЛЬ ПАЛ», имя убийцы и полученная награда. Последний бивший автоматически становится новым Королём с полным HP. Цикл повторяется.
- **Защита Короля:** если Король дарит подарок **сам себе** — лечится (Rose +20 HP, средний +500, большой +2000).

## 🎁 Урон по подаркам (настраивается в `config.js`)

| Подарок | Урон |
|---|---|
| ❤️ Лайк | 5 |
| 🌹 Rose | 10 |
| 💖 Heart | 50 |
| 🎵 TikTok | 100 |
| 🎁 Средние | 250 |
| 💎 Большие | 1000 |
| 🚀 Очень дорогие | 5000 |

## 💰 Награда за голову

Растёт со временем на троне: старт 1000 → 1 мин 2000 → 3 мин 5000 → 5 мин 10000 → 10 мин 25000 → 20 мин 100000 (значения в `config.js`, между точками растёт плавно).

## ✨ Спец-эффекты и события

- 🛡️ **Щит** — иммунитет 15 сек.
- ⚡ **Берсерк** — урон охотников ×2 на 20 сек.
- 💚 **Лечение** — Король восстанавливает 20% HP.
- Случайные события трона (примерно раз в 2,5 мин): 🔥 Восстание (урон ×2), 🛡️ Королевская стража (щит), 💰 Двойная награда (награда растёт быстрее), ⚔️ Час охоты (атаки усилены).

## 🏆 Рекорды и рейтинги

- Самый долгий Король эфира + лучший за день и за неделю.
- 👑 Топ Королей (по времени на троне) и ⚔️ Топ убийц (сколько Королей убил каждый).
- 📜 Живая лента всех событий.

## 🚀 Запуск локально

```bash
npm install
npm start
# открой http://localhost:3000
```

В интерфейсе нажми ⚙️ → введи TikTok `@username` идущего эфира → «Подключить».
Можно задать аккаунт по умолчанию через переменную окружения `TIKTOK_USERNAME`.

### Тест без эфира
- 🎲 **Тест-подарок** — случайный охотник дарит подарок (коронует / бьёт).
- 🎰 **Событие** — запустить случайное событие трона.
- В ⚙️ Настройках: 🛡️ Щит, ⚡ Берсерк, 💚 Лечение, 👑 Король лечится.

## ☁️ Деплой (Render, бесплатно, без карты)

Проект уже настроен под Render (`render.yaml`): runtime Node, build `npm install`, start `npm start`. Render автоматически пересобирает приложение при каждом пуше в GitHub (~2-3 мин).

## 🧩 Технологии

Node.js · Express · Socket.IO · TikTok Live Connector · HTML/CSS/JS. Полная мобильная адаптация. Без базы данных — всё в памяти.

## 📁 Файлы

- `config.js` — все настройки (HP, урон, награда, эффекты, события).
- `game.js` — игровая логика (класс `KingOfLive`).
- `server.js` — Express + Socket.IO + REST API.
- `tiktok.js` — подключение к TikTok Live.
- `public/` — клиент (`index.html`, `style.css`, `app.js`).
